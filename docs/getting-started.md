# Getting started

!!! info "Intro tier"
    The fastest path from zero to a running agentic workflow. Three short steps: install the
    CLI, provision a token, and run your first workflow.

This section walks the **install → provision → compile → commit → run** loop end to end.
Work through it in order the first time; afterward each page stands alone.

1. **[Install `gh aw`](getting-started/install.md)** — add the CLI extension and verify it.
2. **[Provision a token](getting-started/provision-token.md)** — create the PAT, set its
   scopes, and store it as a repository secret.
3. **[Your first run](getting-started/first-run.md)** — author a tiny workflow, compile it,
   commit, and trigger it.
