# Tutorials
{: .no_toc }

{: .tryit }
> Hands-on, end-to-end walkthroughs. Each maps to a sub-proposal and, once built, to a real
> workflow in `.github/workflows/` you can copy into your own repo.

- **[Committer insights](tutorials/committer-insights)** — gather contributor activity with
  Node scripts, then have the agent summarize it into a templated report.
- **[Repository automation](tutorials/repo-automation)** — triage, reporting, and docs
  hygiene built from native Actions and safe outputs.
- **[Tool-using agents](tutorials/tool-using-agents)** — workflows that expose tools (and
  custom MCP servers) for the agent to call in a multi-step loop.

{: .note }
> Stubs for now — the walkthroughs are written as the corresponding workflows are built.
> See the [proposals](https://github.com/titan-syndicate/agentic-workflows/tree/main/docs/proposals)
> for scope.
